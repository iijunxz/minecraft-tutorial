下载并安装Mod¶

下载mod¶

使用mc百科¶

  1. 在MC百科搜索需要的mod

  2. 检查Mod下载，如果不是Mod下载 (0)，则打开并下载

  3. 如果是 Mod下载 (0),则找到相关链接中的CurseForge并打开

  4. 如果没有CurseForge,则尝试其它相关链接

使用CurseForge¶

  1. 首先通过人机验证，通过后一段时间内可以不再次验证

  2. 点击Files,并在打开的页面中点击View All

  3. 找到自己需要的mc版本的对应mod并下载

  4. 如果无法访问，可以选择PCL2或者改hosts

使用PCL2¶

  1. 点击下载按钮

  2. 选择Mod

  3. 在名称中输入需要的mod的名字,并点击搜索

  4. 在搜索结果中找到需要的mod,并下载所需的版本

安装mod¶

无版本隔离¶

将下载完成的Mod放进.minecraft\mods中

有版本隔离¶

将下载完成的Mod放进.minecraft\versions\对应版本\mods中
